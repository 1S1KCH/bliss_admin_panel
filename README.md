Инсталяция:
=================

1) Переименуйте файл adm_config.incRus.php в файл adm_config.php

2) Отредактируйте файл adm_config.php

3) Логин и пароль для входа в админ панель логин <code>superadmin</code> Пароль <code>123456</code>

=================
